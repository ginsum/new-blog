## Blog 2023

- 블로그를 작성하고 수정 가능
- 소개를 보여줄 수 있는 페이지 추가

### 배포 주소

https://hello-hello-bay.vercel.app

### 사용 기술

React, Typescript,  
Tailwindcss, Firebase, Toast-UI, Recoil, React-spring

### 기능

- Toast-UI 사용하여 Markdown 형식으로 글 등록 기능
- Firebase 사용하여 서버 없이 데이터 저장, 로그인 관리
- Tailwindcss 사용하여 스타일 관리
- Recoil 사용하여 User 정보 관리

### 배포

- vercel을 사용하여 배포
